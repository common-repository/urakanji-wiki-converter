== Description ==
This plugin is only supported on PHP 5 or greater.

Urakanji-Wiki-Converter is a text converter.
This plugin is a convert to HTML from plain text written by wiki markup.
so this plugin has possible to use wiki markup in your post.
Reference of wiki markpu was [MediaWiki](http://www.mediawiki.org/wiki/MediaWiki), but not same.

* [Formatting](http://urakanji.com/software/urakanji-converter/formatting/)

== Installation ==

1. Upload all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Write down this word in your post:`__WIKI__`
